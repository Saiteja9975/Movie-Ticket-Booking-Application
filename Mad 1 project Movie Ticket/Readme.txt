Readme for "Movie Ticket Booking App" Project

Introduction:

This project is a Movie Ticket Booking Website that allows users to book tickets  for shows of particular venue. 
Users can browse venues and shows, view details, and book tickets for them. There are two interfaces: one for the users and one for the admin. 
The users can view their bookings , whereas the admin can add,edit and delete venues, shows.

Technologies Used:

Python Flask web framework
Flask SQLAlchemy for database
HTML, CSS, and Bootstrap for the frontend
SQLite3 database to store the data

Instructions to run the code:

Clone or download the repository.
Install the required packages using the command: pip install -r requirements.txt
Run the app.py file using the command: python app.py
Open a web browser and navigate to http://localhost:5000


Explanation of code:
 
The code consists of two Python files: app.py and models.py. The app.py file contains the Flask application and the routes for the different pages.
 The models.py file contains the functions to interact with the SQLite database.
